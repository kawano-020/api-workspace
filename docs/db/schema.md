## 特記カラム属性

- PK : PrimaryKey
- FK : ForeignKey 
- UQ : Uniqe
- NN : Not Null
- AI : Auto Increment
- UN : Unsigned

※ ForeignKey制約には基本的にCASCADEを付与する
